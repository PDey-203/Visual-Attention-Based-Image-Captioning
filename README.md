<h1 align="center" id="title">Visual Attention-Based Image Captioning</h1>

<p id="description">Developed an image captioning pipeline using EfficientNetV2L and LSTM with Bahdanau Attention. Implemented dynamic spatial focusing to improve caption accuracy and contextual relevance by mapping visual features to natural language sequences</p>

  
  
<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Implemented Bahdanau attention to dynamically focus on relevant spatial regions of an image while generating each word of the caption.
*   Used a pretrained EfficientNetV2L CNN backbone to extract high-dimensional visual embeddings from images.
*   Built an LSTM-based language model to convert visual feature vectors into grammatically coherent captions.
*   Mapped spatial image features to textual tokens enabling context-aware caption generation.

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Python
*   TensorFlow/Keras
*   EfficientNetV2L
*   LSTM
